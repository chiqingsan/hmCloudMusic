# hmCloudMusic

#### 介绍
HM云音乐是一款融合音乐与社交的软件，提供关注歌手、评论互动等功能。用户可以关注喜欢的歌手，获取最新动态；在歌曲下方发表评论，分享音乐感悟。此外，还可以查看好友的听歌记录，增进彼此了解。这些社交功能让HM云音乐成为一个音乐爱好者的交流社区。
